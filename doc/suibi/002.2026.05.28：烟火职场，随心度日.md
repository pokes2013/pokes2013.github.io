
# 日常琐记：烟火职场，随心度日

——日常琐记2026.5.28

昨日清晨，照旧早早起身，驱车送李工与魏工前往高铁站。车子平稳行驶的路上，心底无端翻涌着几分复杂的情绪，皆源于公司悄然调整的工时制度。

自老板娘将每日八小时工作制，调整为八个半小时后，办公室里不少人心里都憋着一丝郁结，只是无人轻易言说。我向来随性，能七点半到岗便提前赶来，若是稍赶不及，也会在八点前就位。这份不适从不是我一人独有，方工和几位同事心底皆有怨言，只是大家都默契地保持着沉默。身处人事岗位，我更懂得分寸，即便心有感触，也万万不宜出面直言辩驳。

早前曾与李工商议工时调整事宜，我彼时便提议，办公室作息无需改动，依旧八点上班，适当延长中午午休时间，傍晚五点半便可下班。这般调整既兼顾休息，也不耽误工作，恰到好处。可李工并未应允，只一味顾及部分员工需准时接孩子的诉求，絮絮说了诸多缘由。那一刻心底难免生出几分闷气，职场从来是众人的烟火日常，不该只迁就一方的便利。

思绪浮沉间，车子已然抵达高铁站，时针恰好指向八点十分。距离八点半的发车时间仅剩二十分钟，李工与魏工不敢耽搁，步履匆匆地进站而去。我望着两人的背影，心绪慢慢平复，而后慢悠悠打开导航，驱车折返公司。

回到公司，将车辆停入公车专属车位，心底忽然生出一个念头：想和华英负责车位管理的小陶沟通，将自己的车牌号录入公车车位系统，日后停车也能便利些。可思忖再三，终究还是压下了这个想法。

我素来对华英办公室的一众人员没什么好感，早前带着员工办理宿舍入住时，负责宿舍管理的那位大姐态度凌厉、语气生硬，那段不甚愉快的经历，让我不愿再多打交道，索性安于现状，不多强求。

正午十一点半，我和老婆下楼就餐，途中偶遇老王。他随口嘱托，让我下午陪同他前往市政府参会。我心底着实万般不情愿，昨夜休息不佳，周身酸软乏力，整个人都透着疲惫，只想静静休憩。耐不住老王再三劝说，一句“反正无事，权当出门走走”，终究还是应了下来。

午饭后难得清闲，我枕着暖意沉沉小憩，一直睡到下午一点二十五分。直到王工走到工位唤我，才恍然回神，收拾妥当后一同下楼驱车赴会。刚睡醒的头脑格外清明，便由我驾车前行。导航显示全程四公里，不算遥远，却因市区路况，预估需半小时方能抵达。最是费时的是找车位，兜兜转转二十余分钟，才终于觅得空位停妥车辆。

停好车后，我们跟着往来人流步入市政大楼，直达三楼会场。到场之人大多是企业老板、部门经理级别的人物，男女参差，电梯间里萦绕着各式淡雅浓郁的香水气息，混杂出职场独有的氛围。

出电梯便是签到台，三张清丽的前台小姐姐立在桌前，井然有序地接待着到场宾客。我与老王俯身对着签到表细细翻看，足足寻觅了五分钟，才找到我们公司的名字。前台见状有些诧异，轻声问道：“你们是两位吗？本次席位只预留了一个，另一位可以坐洽谈区。”

翻看席位图，我们的正式席位在第一列第三行。入场落座前，老王转头问我：“你去前排就座，还是我去？”我连忙推脱，让他前去落座。一来是心底怯懦，生怕临场被点名发言、接受采访，难免局促窘迫；二来事后听李工说起，本次会议发言环节皆是提前预定，是我多虑了。更重要的是，洽谈区格外自由，无需拘谨端坐，可随意起身走动，也能出门透气抽烟，反倒自在松弛。

会议正式启幕，由副市长致开幕词，而后各家企业负责人轮番上台，借着PPT推介自家的产品与服务。我独坐清闲的洽谈区，起初偶尔翻看手机，每当摄像、拍照工作人员从身侧走过，便连忙收起手机，正襟危坐，佯装认真聆听会议内容，生怕被镜头捕捉到闲散模样。

会议持续了一个半小时左右，场内轮番的宣讲声不绝于耳，听得人些许乏味。我悄然移步走出会场，踱步至吸烟区，点燃一支烟静静放空。心底不由生出几分疑惑，这场名为供应链交流会的活动，全程却只剩各家企业轮番推介、自我推销，早已偏离了交流互通的初衷，隐隐变了味道。也难怪老板不愿到场，特意委派王工代为参会，想来早已洞悉其中端倪。

烟尽神舒，走进市政府干净敞亮的洗手间，简单休整片刻，便想着下楼买瓶水解渴。会场内虽备有饮用水，却并非一次性水杯，皆是统一的白色陶瓷杯。即便看着洁净无垢，我也未曾动念，一来身处洽谈区，不便随意取用；二来多年前一则陶瓷水杯的负面新闻始终萦绕心头，心底芥蒂难消，索性作罢。

走出市政大楼，抬眼望去，道路两侧商铺林立，尽是五金建材门店，满目烟火景致，果然不负永康“五金之都”的盛名。我顺着路边缓缓右行闲逛，行至红绿灯路口，望见马路对面有一间便民小店，便打算过去买水。

路口车流规整、秩序井然，到底是市政府周边，交通风貌尚且体面。转念想起永康寻常路段的交通乱象，心底不免感慨。此前我曾实在忍无可忍，在小红书悄悄吐槽过此地交通：车辆随意停放、转弯车辆不让直行、电动车肆意穿行马路中央，鸣笛提醒非但无人理会，反倒招致侧目谩骂。有趣的是，帖子下有人留言，直言大环境如此、无可奈何，我私下总暗自揣测，留言之人或许便是交管相关工作人员。

进店选水之际，老王大概是耐不住会场的枯燥，也脱身出来，打电话询问我的位置。我告知他在附近买水，让他在车旁稍作等候。我素来偏爱茶饮，自上次广州之行后，便常喝东方树叶，于是给自己拿了一瓶，又为老王挑选了一瓶脉动。

步行折返的十余分钟里，我暗自思忖说辞，缓解自己提前离场的些许尴尬。见到老王后，我笑着坦言，这场交流会早已变味，全无互通交流的意义，反倒成了各家的产品推介会，实在无趣。

老王闻言会心一笑，坦言难怪老板不愿亲自到场。两人闲谈说笑间，陆续有不少参会人员走出会场，想来大家心境大抵相同，都觉这场会议索然无味。平凡职场一日，便在这般细碎感慨与松弛闲谈中，缓缓落幕。
