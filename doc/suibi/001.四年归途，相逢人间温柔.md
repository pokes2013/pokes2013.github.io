
# 四年归途，相逢人间温柔
——感概记忆2019

一晃经年，距离上一次踏回故土，已是整整四年。

犹记2019年八月，归乡的脚步沉重又仓促，那次回家，只为一场开庭。人生的风雨，猝不及防砸在肩头，婚姻离散，心底满是难堪与落寞。总觉得离婚是件抬不起头的事，便刻意屏蔽归途，把自己困在异乡的方寸天地里，一躲，便是四年。

这四年，横跨了疫情三年的漫长岁月。我在时光里悄悄蜕变，熬过了最灰暗的日子，考下了驾照，添置了爱车，换了新的发型，也慢慢抚平了心底的褶皱。

曾经那段破碎的婚姻，让我颓废沉沦了大半年。满心的委屈、不甘与怨恨，尽数倾泻在心底，恨过那个人，也怨过那一大家子。情绪压抑到极致时，心态几近扭曲，但凡生活、工作稍有不顺，心底便会生出极端的念头，甚至有过偏激冲动。那段日子，像是被困在黑暗的牢笼里，找不到出口。

后来不愿再任由自己沉沦，便报名学车。旁人说，学车学的不仅是技术，更是心态。深以为然。科目三不慎挂科一次，其余科目皆是一次通关。当拿到驾照的那一刻，心底忽然豁然开朗。原来过往的爱恨纠葛、失意坎坷，都算不上人生的劫难，不过是岁月赋予的一段经历而已。往事皆序章，人总要学会和过去和解，学着向阳前行。

再后来买了车，日子慢慢安稳下来，心境也日渐平和。只是受过情伤，便很难再轻易相信感情，也从未有过再踏入婚姻的打算，只想安稳度日，独自安好。

今年二月，家里开始张罗着为我介绍相亲。遇见过功利的人，一开口便索要房车，要求房贷全无，还要五十万启动资金。这般不切实际的期许，更像是凭空许愿，哪里是寻觅伴侣，分明是依附索取。我果断拉黑，反倒落得一身清净。

四月伊始，遇见了如今的她。初见相处格外安静，我们都是性子内敛的老实人，隔着屏幕视频，常常两两相望，无言也不觉得尴尬。就这样静静闲聊，一来一往，便是一个月。

酝酿许久，终于决定趁着五一归途，奔赴家乡，见一见朝思暮想的人，也好好陪伴久未相见的父母。

4月28日清晨九点，列车驶入老家车站。远远便看见父亲等候的身影，猛然发觉，岁月从不饶人，父亲的发丝早已染满霜白。他开着那辆白色吉利手动挡，载着我踏上回家的路。阔别四年，故土的风依旧熟悉，只是常年在外，终究难免水土不服，归家后竟莫名腹泻，大抵是游子身心，早已习惯了异乡烟火。

休整过后，午后三点漫步新建路，小城早已换了新颜，四处都是陌生又亲切的模样。顺路走进街边小店，随手备了一个红包，包上两百元，权当给孩子的见面心意。

傍晚四点半，终于迎来初次见面。四年未归，小城街巷变迁，我竟一时分不清方向，阴差阳错走到了实验中学。初见之时，两人皆是满心拘谨，带着几分成年人的羞涩。她带着孩子，懂事的小家伙独自跑去操场玩耍，留我们二人在跑道边缓步闲谈。没有太多华丽的言辞，轻声慢语，皆是平实的家常，安静，却格外心安。

傍晚六点，寻思着找一处地方吃饭，茫然无措间，拨通了老友的电话，最终选定上河郡的串串香火锅店。我牵着孩子挑选食材，特意加了一份羊肉，后来才知晓，她自幼吃肉伤了脾胃，自此便再不食荤腥。那日她身着简约白外套，搭配时下流行的开叉灰色卫裤，干净素雅，温婉大方。

我们闲谈小坐，从日暮吃到夜色渐浓。九点走出火锅店，母亲打来电话，叮嘱我去超市买菜。她执意要陪我同去，推脱不过，便并肩走进商超。挑选完家常食材，孩子念叨着想吃榴莲，我便挑了一个硕大的榴莲，予孩子一份欢喜。

送她们离去后，我独自归家，到家时已是夜里十点多。故土的夜晚静谧温柔，晚风拂面，吹散了心底多年的孤寂。

4月29日，归家的第二日。媒人同父亲商议，我难得回乡一趟，不如邀请她的母亲来家中小聚，也算正式相见。清晨九点，我拨通了她的电话，起初打算同父亲一同前去登门拜访，斟酌过后又觉不妥。父亲便嘱咐我，开车去接她先来家里小坐。

这是我第一次上手开父亲的手动挡车子，许久不练，手法生疏，一路上频频熄火，磕磕绊绊，总算在十点左右安稳将她接到家中。那日她一身黑色简约套装，利落精致，身段窈窕，格外耐看。虽是历经世事的过来人，可我望着她，依旧心生腼腆，不敢细细打量，只安静陪在一旁，陪着孩子嬉戏打闹，时光慢悠悠，温柔又惬意。

午后两点，父亲同媒人也将她的母亲接至家中。初见长辈，我略显局促，轻声唤了一声阿姨。母亲转身入厨忙碌，我便在一旁帮忙端菜摆桌，满屋烟火气，皆是家常温情。

闲谈数小时后，傍晚时分，一行人再赴上河郡设宴，算作正式的相亲小宴。父母备下了见面礼，礼数周全，心意诚恳。前一日我给孩子备好的红包，也悄然递了过去。

宴席散去，我陪她重回我家，父母随后送长辈返程。夜色渐沉，我驱车送她回家。楼道窗边，我们静静并肩闲谈，晚风透过窗棂漫进来，岁月静好，相逢恰逢温柔。

四年漂泊，终归故里。放下过往执念，告别心底阴霾，在家乡的烟火里，遇见合拍的人，陪伴年迈的父母。原来人生最好的归宿，不过是叶落归根，有人可念，有家可回，有温柔可相逢。
