## IP地址的组成   

由四组8位bit得二进制位数组成，每组8位bit，共计32位
例如：0.0.0.0
```bash  
00000000  00000000  00000000  00000000  
```
我们肉眼看到得IP地址是10进制，计算机看到得是二进制

## IP地址的分类   

- A类    0    第一位代表0-127；      
- B类    10    第一位代表128，范围为129-191；      
- C类    110    第一位任然代表128，第二位为64，所以是128+64=192，范围：192.0.0.0-223.255.255.255

## IP地址的特点
- A类地址，例如：58.211.107.14，其中58是网络号，211.107.14是主机地址，所能容纳得主机数量为：255x255x255=16581375
- B类地址，例如：129.16.10.12，其中129.16是网络号，10.12是主机地址，所能容纳得主机数量为：255x255=65535
- C类地址，例如：192.168.10.12，其中192.168.10是网络号，12是主机地址，所能容纳得主机数量为：255
