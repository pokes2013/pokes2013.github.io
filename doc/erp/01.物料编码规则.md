# 编码和命名规则

## 一、物料编码规则的总体结构

编码结构为 **12 位全数字**，共分为 5 段，具体结构如下：

```plaintext
X    X X     X X X    X X X X X X
种类  大类   三级分类  规格型号+流水号
```

各分段详细说明：

1. **种类**：1 位，可扩展 9 种物料种类示例：

   - 1：成品
   - 2：半成品
   - 3：原材料
   - 4：零部件
   - 5：包材
   - 6：标准件
   - 7：样品
   - ……

2. **大类**：2 位，在某一种类下可扩展 99 种细类示例 1（成品类）：

   - 101：有刷电机 Y

   - 102：无刷电机 W

   - 103：电控版 D

   - 104：保护板 B

   - ……

     

     示例 2（原材料类）：

   - 301：有刷原材料

   - 302：无刷原材料

   - 303：电控原材料

   - ……

3. **三级分类**：3 位，在某一大类下可扩展 999 种细类示例 1（有刷电机 101）：

   - 101001：有刷电机 3 系类

   - 101002：有刷电机 5 系类

   - 101003：有刷电机 6 系类

   - ……

     

     示例 2（原材料 301）：

   - 301001：有刷机壳轴承类

   - 301002：有刷磁瓦类

   - 301003：有刷磁环类

   - ……

     

     示例 3（零部件 401）：

   - 401001：有刷机壳类

   - 401002：有刷转子类

   - 401003：有刷端盖类

   - ……

     

     示例 4（包材 501）：

   - 501001：有刷纸箱类

   - 501002：有刷吸塑盘类

   - ……

4. **规格型号**：6 位，将物料名称关键信息纳入编码，不足 6 位时前面补 0编码规则：

   1. 优先以产品型号数字和主要区分参数编码

      示例 1：普通级有刷 997 型电机 / HP——20V/23000 转 / 轴长 104mm / 线径 0.75mm/20 圈 / 带电容 → 完整编码：101001010001

      示例 2：550/75x30x24.8/13.9 / 窄 /Φ23.0 / 高碳钢轴 → 完整编码：401002020001

   2. 零部件类以技术参数数值编码

      示例 1：有刷 5 系轴承；贴滚珠紧配 /Φ12.0xΦ3.175/-7-12/xT4 / 不锈钢 → 完整编码：601003020001

      示例 2：有刷 5 系垫片：波形垫片 /Φ6.0xΦ3.18xT0.15xH0.85 / 铍铜 → 完整编码：601006020001

      注：规格型号编码无需刻意记忆规则

5. **流水码**：用于编码规则的无限扩展，提升灵活性

## 二、物料命名方法

### 1. 电机类物料命名要求

电机名牌需包含以下信息：电机型号、额定功率、额定电压、额定频率、额定转速、防护等级、绝缘等级、制造厂家、出厂日期

### 2. 命名示例

示例 1（电机类）：

- 现有名称：SH997-20180-122C-001
- 现有规格：SH997-75007-122C-001（项目号 料号 客户代码）
- 规范名称：普通级（S 级）有刷 9 系 997 型电机 / 客户代码（HP）
- 规范规格：20V/**W/23000 转 / 轴长 104mm / 线径 0.75mm/20 圈 / 带电容

示例 2（线材类）：

- 现有名称：铜线 / 漆包线
- 现有规格：φ0.67mm QZY-2/180°
- 规范名称：铜线 0.67 / 漆包线 0.67
- 规范规格：180°/φ0.67mm/QZY-2

## 三、注意事项

1. **名称统一**：同一类物料名称需固定，例如 “铜线” 不能随意变更为 “转子铜线” 等其他名称。

2. **排列顺序统一**：规格参数的排列顺序需一致，否则系统会判定为不同物料。

   

   反例：

   ```
   铜线0.67—180°/φ0.67mm/QZY-2
   ```

    

   与

    

   ```
   铜线0.67—φ0.67mm/180°/QZY-2
   ```

    

   会被识别为两种物料。

## 四、物料编码完整示例

| 编码项   | 内容                                                       |
| -------- | ---------------------------------------------------------- |
| 物料编码 | 101002010001                                               |
| 物料名称 | 普通级（S 级）有刷 9 系 997 型电机 / HP                    |
| 物料规格 | 20V/**W/23000 转 / 轴长 104mm / 线径 0.75mm/20 圈 / 带电容 |
| 客户编码 | 现系统中料号                                               |
| 客户名称 | 现系统对外名称或客户要求名称                               |
| 客户规格 | 现系统对外规格或客户要求规格                               |
| 项目号   | 保持不变                                                   |

**优化建议**：采用 `种类+大类+三级分类+4位流水号` 的结构更简洁，规范物料命名方法即可规避一物多码的问题。

# 销售订单号规则

## 一、订单号示例

1. 纯日期 + 流水号格式：24082801、24082802、24082803、24082803-1、24082803-2
2. 日期 + 跟单员标识 + 流水号格式：2408C001、2408SN01、2408Y01、2408MG01、2408Z01、2408XM01、2408L01、2408YC01

## 二、编码规则

订单号结构：**年 (2 位)+ 月 (2 位)+ 跟单员标识 + 流水号 (001-999)**

### 1. 跟单员标识对应表

| 标识 | 跟单员 |
| ---- | ------ |
| A    | 曹     |
| B    | 颜     |
| C    | 郑     |
| D    | 刘     |

### 2. 完整示例

订单号：2408C001含义：2024 年 08 月 郑秀梅 第一单
