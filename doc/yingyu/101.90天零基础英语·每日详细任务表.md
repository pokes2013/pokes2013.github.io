## 第1～30天｜打地基（每天30分钟）
### 每天固定流程：
1. **音标 10分钟**
   - 目标：48个国际音标，会读、会区分
   - 每天学 **2～3个**，复习前面所有
   - 会读就行，不用背名词

2. **单词 10分钟**
   - 每天 **10个** 基础词
   - 必背类别：
     - 数字 1～100
     - 星期、月份
     - 家人、职业
     - 常见动词：go, come, do, make, like, want, work
   - 要求：会读 + 知道中文

3. **跟读句子 10分钟**
   每天只练这5句，读到顺口：
   - I'm ... 我是……
   - My name is ... 我叫……
   - I like ... 我喜欢……
   - I can ... 我会……
   - This is ... 这是……

**30天结束你能做到：**
会自我介绍、会读单词、敢开口。

---

## 第31～60天｜语法+句型（每天30分钟）
### 每天固定流程：
1. **语法 10分钟**
   只学这6个，每天学1个，循环复习：
   1. am / is / are
   2. 一般现在时
   3. 现在进行时（am/is/are doing）
   4. 一般过去时
   5. 一般将来时（will / be going to）
   6. can / want / like 用法

2. **句型 10分钟**
   每天背 **3个万能句型**，套词造句：
   - I want to ... 我想……
   - I'm going to ... 我要……
   - I can ... 我会……
   - I need ... 我需要……
   - How do you ...? 你怎么……？

3. **听力跟读 10分钟**
   - 听 **超短句**，一句一句暂停跟读
   - 不求多，只求**听得清、跟得上**

**60天结束你能做到：**
造简单句基本不错，能听懂慢速日常对话。

---

## 第61～90天｜听说读写起飞（每天30分钟）

### 每天固定流程：
1. **听力 10分钟**
   - 日常对话、1分钟以内短视频
   - 能听懂大意就行，不抠每个词

2. **口语 10分钟**
   每天**自言自语3句话**：
   - What I did today 我今天做了什么
   - What I want to do 我想做什么
   - How I feel 我感觉怎么样

3. **阅读+写作 10分钟**
   - 读5～10句简单英文
   - 写3～5句日记（用你学过的词）

**90天结束你能做到：**
日常简单交流没问题，能看懂工作里的简单英文。
