# 3.2.对象

## 对象的创建

```java
package demo01;

public class Student{
    String name;
    int age;

    public void study() {
        System.out.println("我叫" + this.name + "我今年" + this.age + "岁了");
    }
}



package demo01;

public class test06 {

    public static void main(String[] args) {
        Student xm =new Student();
        xm.name = "小明";
        xm.age = 13;
        xm.study();		//我叫小明我今年13岁了

    }
}
```
