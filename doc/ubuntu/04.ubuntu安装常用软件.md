# ubuntu安装常用软件

## 安装浏览器

```bash
# 安装依赖工具+导入谷歌官方GPG密钥
sudo apt update && sudo apt install -y wget gnupg && wget -q -O - https://dl.google.com/linux/linux_signing_key.pub | sudo gpg --dearmor -o /usr/share/keyrings/google-chrome-keyring.gpg
# 添加谷歌浏览器官方软件源
echo "deb [arch=amd64 signed-by=/usr/share/keyrings/google-chrome-keyring.gpg] https://dl.google.com/linux/chrome/deb/ stable main" | sudo tee /etc/apt/sources.list.d/google-chrome.list
# 更新源并安装最新稳定版谷歌浏览器
sudo apt update && sudo apt install -y google-chrome-stable
```

## 输入法切换不了中文

```bash
# 配置IBus输入源（Ubuntu22.04默认）
gsettings set org.gnome.desktop.input-sources sources "[('xkb', 'us'), ('ibus', 'libpinyin')]"
# 设置切换输入源快捷键为Ctrl+空格
gsettings set org.gnome.desktop.wm.keybindings switch-input-source "['<Control>space']"
gsettings set org.gnome.desktop.wm.keybindings switch-input-source-backward "['<Shift><Control>space']"
```



## vpn代理
