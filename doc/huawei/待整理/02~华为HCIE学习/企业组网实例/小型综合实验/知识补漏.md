# 知识补漏

## 1、NAT配置

```
[ar1]acl 2000
[ar1-acl-basic-2000]rule permit source 192.168.0.0 0.0.255.255
[ar1-acl-basic-2000]q
[ar1]nat address-group 1 64.1.1.5 64.1.1.5  //使用64.1.1.5来做nat，不能使用64.1.1.1因为被接口已经占用了。
[ar1]int g0/0/1
[ar1-GigabitEthernet0/0/1]nat outbound 2000 address-group 1
```

## 2、PC1不允许上网

```
[ar1-acl-basic-2001]rule deny source 192.168.10.0 0.0.0.255   //不允许PC1段
[ar1-acl-basic-2001]rule permit source any   //允许其他
[ar1-acl-basic-2001]	
[ar1-acl-basic-2001]
[ar1-acl-basic-2001]q
[ar1]int g0/0/0
[ar1-GigabitEthernet0/0/0]traffic-filter inbound acl 2001   //调用acl2001
```

## 3、内网服务器发布到外网

```
[ar1-GigabitEthernet0/0/1]nat server global 64.1.1.3 inside 172.168.100.1 
```
