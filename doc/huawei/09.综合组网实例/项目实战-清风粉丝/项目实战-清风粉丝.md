## 核心交换机SW1配置

### 划分vlan

vlan10,员工上网使用，需要开启DHCP，网段192.168.10.0/24

vlan20服务器使用，不需要DHCP，网段192.168.20.0/24

vlan30给AC使用，不需要DHCP，也不需要太多的IP地址，网段192.168.30.0/29，范围192.168.30.1-192.168.30.6

vlan200用于链接AR1主路由器，也不需要太多的IP地址，网段192.168.200.0/29，范围192.168.200.1-192.168.200.6

```
[Huawei]sy hexin
[hexin]un in en
Info: Information center is disabled.
[hexin]vlan batch 10 20 30 200
[hexin]int g0/0/1
[hexin-GigabitEthernet0/0/1]port lin acc
[hexin-GigabitEthernet0/0/1]po de vlan 10
[hexin-GigabitEthernet0/0/1]int g0/0/5
[hexin-GigabitEthernet0/0/5]po lin acc
[hexin-GigabitEthernet0/0/5]po de vlan 30
[hexin-GigabitEthernet0/0/5]int g0/0/3
[hexin-GigabitEthernet0/0/3]po lin acc
[hexin-GigabitEthernet0/0/3]po de vlan 20
[hexin-GigabitEthernet0/0/3]int g0/0/4
[hexin-GigabitEthernet0/0/4]po lin acc
[hexin-GigabitEthernet0/0/4]po de vlan 20
[hexin-GigabitEthernet0/0/24]po lin acc
[hexin-GigabitEthernet0/0/24]po de vlan 200

#在vlan10上面开启“给予接口的DHCP”

[hexin]int vlan10
[hexin-Vlanif10]dis th
#
interface Vlanif10
 ip address 192.168.10.1 255.255.255.0
 dhcp select interface
 dhcp server dns-list 192.168.20.12 114.114.114.114      #192.168.20.12是一个域控制器，我们把他配置成首选dns

[hexin]int vlan 30
[hexin-Vlanif30]ip add 192.168.30.1 29

[hexin]int vlan 20
[hexin-Vlanif20]ip add 192.168.20.1 24
[hexin-Vlanif20]q
```

这样核心交换机基础配置就已经配置完成了。

## AC的基础配置



```
[Huawei]sy ac
[ac]un in en
Info: Information center is disabled.
[ac]
[ac]int g0/0/1
[ac-GigabitEthernet0/0/1]ip add 192.168.30.2 29
[ac-GigabitEthernet0/0/1]q
[ac]int g0/0/2
[ac-GigabitEthernet0/0/2]ip add 192.168.90.1 24
[ac-GigabitEthernet0/0/2]

#开启DHCP

[ac]dhcp enable 
Info: The operation may take a few seconds. Please wait for a moment.done.
[ac]ip pool pokes
Info: It's successful to create an IP address pool.
[ac-ip-pool-pokes]network 192.168.90.0 mask 24
[ac-ip-pool-pokes]gateway-list 192.168.90.1	
[ac-ip-pool-pokes]dns-list 192.168.20.12	
[ac-ip-pool-pokes]dns-list 114.114.114.114	
[ac-ip-pool-pokes]lease day 0 hour 20   		  #租约
[ac-ip-pool-pokes]static-bind ip-address 192.168.90.100 mac-add 5489-98E7-76DF   #绑定静态地址
[ac-ip-pool-pokes]excluded-ip-address 192.168.90.200 192.168.0.254    #保留地址
[ac-ip-pool-pokes]dis th
[V200R003C00]
#
ip pool pokes
 gateway-list 192.168.90.1 
 network 192.168.90.0 mask 255.255.255.0 
 static-bind ip-address 192.168.90.100 mac-address 5489-98e7-76df 
 lease day 0 hour 20 minute 0 
 dns-list 192.168.20.12 114.114.114.114 
 excluded-ip-address 192.168.90.200 192.168.0.254
#
return

```

## 主路由的基础配置

```
[Huawei]sy route
[route]un in en
Info: Information center is disabled.
[route]int g0/0/0
[route-GigabitEthernet0/0/0]ip add 192.168.200.1 29
[route-GigabitEthernet0/0/0]
```

## 加路由

```
#AR1主路由上面配置，到内外192.168.10/20/30/90.0段的回执路由
[route]ip route-static 192.168.0.0 16 192.168.200.2

#在核心交换机sw1上面加入默认路由
[hexin]ip route-static 0.0.0.0 0 192.168.200.1

#在核心交换机sw1上面加入到达AC的回执路由
[hexin]ip route-static 192.168.90.0 24 192.168.30.2

#在AC上面加默认路由
[ac]ip route-static 0.0.0.0 0 192.168.30.1
```
