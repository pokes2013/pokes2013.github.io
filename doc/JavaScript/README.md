### 一、入门阶段（1-2 个月）：夯实三驾马车基础

**核心目标**：掌握前端基础三件套，能独立实现静态页面与简单交互

| 技术模块            | 核心学习内容                                                 | 实践项目                       | 时间规划 |
| ------------------- | ------------------------------------------------------------ | ------------------------------ | -------- |
| **HTML5**           | ・语义化标签（header/nav/article/section）・表单与表单验证・多媒体标签・新增属性与 API | 个人简介页企业官网静态版       | 3-4 周   |
| **CSS3**            | ・选择器优先级与权重・盒模型与 BFC・Flex+Grid 布局（重点）・响应式设计 + 媒体查询・过渡与动画・CSS 变量与嵌套 | 响应式博客移动端电商首页       | 3-4 周   |
| **JavaScript 基础** | ・变量 / 数据类型 / 运算符・流程控制与函数・数组与对象操作・DOM/BOM 操作・事件冒泡 / 捕获・ES6 + 基础（解构 / 箭头函数） | 待办事项列表轮播图组件表单验证 | 5-7 周   |

**关键产出**：完成 3 个静态页面 + 2 个交互组件，代码量≥**5000 行**



### 二、进阶阶段（3-6 个月）：框架与工程化能力

**核心目标**：掌握主流框架、前后端交互与工程化流程，能独立开发中大型项目

#### 1. JavaScript 核心进阶（4 周）

- 原型链与继承机制
- 闭包与作用域
- 异步编程（Promise/async-await）
- 事件循环与宏 / 微任务
- 防抖节流与性能优化
- 模块化（ES Module/CommonJS）

#### 2. 前端框架（选 1 个深耕，推荐 Vue3 优先）（6 周）

- **Vue3**：Composition API、组件通信、Vue Router、Pinia、Teleport/Suspense
- **React**：Hooks、状态管理、React Router、Context API、性能优化
- **项目实战**：管理系统后台（含登录 / 权限 / 表格 / 表单）

#### 3. 工程化工具链（4 周）

- 包管理：npm/yarn/pnpm
- 构建工具：Vite（推荐）/Webpack
- 代码规范：ESLint+Prettier
- 版本控制：Git+GitHub/GitLab
- 类型安全：TypeScript 基础

#### 4. 前后端交互（4 周）

- HTTP/HTTPS 协议
- AJAX 与 Fetch API
- Axios 封装与拦截器
- RESTful API 设计
- 跨域解决方案（CORS / 代理）

**关键产出**：1 个完整 SPA 项目（如电商后台），代码量≥**20000 行**，使用**Git 管理**

------

### 三、精通阶段（6-12 个月）：底层原理与性能优化

**核心目标**：深入前端底层，解决复杂技术问题，具备架构设计能力

#### 1. 底层原理（8 周）

- 浏览器渲染原理（重排 / 重绘）
- 虚拟 DOM 与 diff 算法
- 编译原理（AST）
- 前端安全（XSS/CSRF）
- 网络优化（HTTP 缓存 / CDN）

#### 2. 高级技术栈（8 周）

- TypeScript 进阶（泛型 / 类型体操）
- 状态管理进阶（Vuex/Redux/Zustand）
- 组件库开发（组件设计 / 封装）
- 前端工程化进阶（CI/CD/ 自动化测试）
- 微前端（可选）

#### 3. 性能优化（4 周）

- 代码分割与懒加载
- 图片优化（WebP/SVG）
- 首屏加载优化
- 长列表性能优化
- 内存泄漏排查

#### 4. 全栈开发入门（8 周）

- Node.js+Express/Koa
- MongoDB/MySQL 基础
- 服务端渲染（SSR/SSG）
- Serverless 基础
- 项目实战：个人博客系统（前后端 + 数据库）

**关键产出**：1 个全栈项目，掌握**2 种**性能优化方案，撰写技术博客≥**10 篇**

### 五、学习资源与时间规划表

#### 最佳学习资源

1. **文档类**：MDN Web Docs、Vue/React 官方文档、TypeScript Handbook
2. **视频类**：黑马程序员、尚硅谷、B 站前端教程
3. **练习平台**：LeetCode（算法）、CodeSandbox（快速原型）
4. **社区**：掘金、知乎、GitHub、Stack Overflow

#### 每周学习计划（建议）

- 基础阶段：每日编码≥**3 小时**，代码量≥**300 行**
- 进阶阶段：每日编码≥**4 小时**，每周提交**3 次 Git**
- 精通阶段：参与开源项目，每周技术输出≥**1 篇**

### 行动清单（立即执行）

1. 安装 VS Code + 插件（ESLint/Prettier/Vetur）
2. 搭建 GitHub 账号，创建学习仓库
3. 每周制定明确目标，代码提交≥**3 次**
4. 每月重构 1 个旧项目，持续优化代码质量
5. 加入前端社区，定期分享学习心得

**路线图总结**：前端学习遵循 “**基础→框架→工程化→全栈→前沿**” 的递进路径，核心在于**持续编码 + 项目驱动 + 复盘总结**。坚持**1 年**，你将具备初级前端工程师能力；坚持**2 年**，可成长为中级工程师；**3 年 +** 深耕，可成为技术专家或全栈工程师。
