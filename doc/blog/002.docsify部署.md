# docsify部署


官网：https://docsify.js.org/#/

下载nodejs：https://www.nodejs.com.cn/download.html

我这里使用的nodejs18.19.0长期维护版本

---

安装nodejs18.19.0一直到下一步完成.

```powershell
PS C:\Users\Anita> node -v
v18.19.0
```

全局安装docsify

```powershell
npm i docsify-cli -g
```
