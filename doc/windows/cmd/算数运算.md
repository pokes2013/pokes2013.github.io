# 算数运算

## 算数运算

加+，减-，乘*，除/，取余%

基本用法：

```bash
C:\Users\Anita>set /a 1+2
3
C:\Users\Anita>set /a 2*2
4
C:\Users\Anita>set /a 10/5
2
C:\Users\Anita>set /a 10/3    #除法只显示整数
3
C:\Users\Anita>set /a 10%3    #取余数
1
C:\Users\Anita>
```

脚本实例：

```bash
@echo off

set /a var = 1 + 2
echo %var%

pause

##关于计算的优先级和我们数学里面是一样的加括号，先算括号内的。
@echo off

set /a var = 10*(9+ 2)
echo %var%

pause
```
