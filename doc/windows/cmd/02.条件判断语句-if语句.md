# 02.条件判断语句-if语句

## 判断的基本规则

1、判断文件或文件夹是否存在，用if exist 语句

2、判断某两个字符串是否相等，用if "字符串1"=="字符串2” 语句

3、判断某个变量是否已经被赋值，用if defined str 语句

4、判断某两个数值是否相等，用if 数值1 equ 数值2 语句

‍

|中文含义|关系符|英文解释|
| ---------------------------| --------------------| --------------------------------------|
|等于|equ|equal|
|大于|gtr|greater than|
|大于或等于|geq|greater than or equal|
|小于|lss|less than|
|小于或等于|leq|less than or equal|
|不等于|neq|no equal|

‍

## 判断是否存在

可用用于判断文件、目录、盘符等

exist和not exist

```Python
@echo off
if exist F:\ (echo 存在F盘）else (echo 不存在F盘) 
pause >nul

上面的代码取反
@echo off
if not exist F:\ (echo 存在F盘）else (echo 不存在F盘) 
pause >nul
```

## 判断字符串相等

```Python
@echo off
if  "hello"=="hello"  (echo equql)  else  (echo not equal) 
pause >nul

注意：默认是区分大小写的。
忽略大小写：if  /i "hello"=="hello"  (echo equql)  else  (echo not equal) 
```

## 数字比大小

```Python
@echo off
set varl=hello
set var2=hello
if %var1%==%var2% (echo equal) else (echo not equal) 
pause >nul
```

接受用户输入比大小

```Python
@echo off
set /p varl=请输入比较的字符串1：
set /p var2=请输入比较的字符串2：
if %var1%==%var2% (echo equal) else (echo not equal) 
pause >nul
```
