# 正则表达式re模块使用大纲
## 01 re模块概述
re模块是Python内置模块，无需安装，用于实现正则表达式相关操作

## 02 re模块核心功能函数
| 函数                                      | 功能描述                                                     |
| ----------------------------------------- | ------------------------------------------------------------ |
| re.match(pattern,string,flags=0)          | 从字符串开始位置匹配，匹配成功返回Match对象，否则返回None     |
| re.search(pattern,string,flags=0)         | 在整个字符串中搜索第一个匹配值，匹配成功返回Match对象，否则返回None |
| re.findall(pattern,string,flags=0)        | 在整个字符串中搜索所有符合正则表达式的值，返回列表类型       |
| re.sub(pattern,repl,string,count,flags=0) | 实现对字符串中指定子串的替换                                 |
| re.split(pattern,string,maxsplit,flags=0) | 分隔字符串，功能与字符串的split()方法相同                    |

## 03 re.match 匹配开头
### 3.1 语法
```python
re.match(pattern,string,flags=0)
```
### 3.2 特点
从字符串起始位置匹配，起始位置不匹配则返回None
### 3.3 示例
```python
import re
# 匹配规则
pattern = "\d\.\d+"
# 待匹配字符串
s1 = "I study Python 3.11 every day"
match1 = re.match(pattern, s1, re.I)
print(match1)  # 结果：None

s2 = "3.11Python I study every day"
match2 = re.match(pattern, s2, re.I)
print(match2)  # <re.Match object; span=(0, 4), match='3.11'>
```
### 3.4 Match对象常用方法
- start()：获取匹配值的起始位置
- end()：获取匹配值的结束位置
- span()：获取匹配区间的位置元组
- string：获取待匹配的原始字符串
- group()：获取匹配到的数据

## 04 re.search 搜索匹配
### 4.1 语法
```python
re.search(pattern,string,flags=0)
```
### 4.2 特点
在整个字符串中搜索第一个匹配值，匹配到后立即结束
### 4.3 示例
```python
import re
pattern = "\d\.\d+"
s1 = "I study Python 3.11 every day Python2.7 I love you"
match = re.search(pattern, s1)
print(match)  # <re.Match object; span=(15, 19), match='3.11'>
print(match.group())  # 3.11
```

## 05 re.findall 查找所有匹配
### 5.1 语法
```python
re.findall(pattern,string,flags=0)
```
### 5.2 特点
搜索整个字符串中所有符合规则的值，返回列表类型
### 5.3 示例
```python
import re
pattern = "\d\.\d+"
s1 = "I study Python 3.11 every day Python2.7 I love you"
result_list = re.findall(pattern, s1)
print(result_list)  # ['3.11', '2.7']
```

## 06 re.sub 字符串替换
### 6.1 语法
```python
re.sub(pattern,repl,string,count,flags=0)
```
### 6.2 特点
替换字符串中符合正则规则的子串
### 6.3 示例
```python
import re
pattern = "黑客|破解|反爬"
s1 = "我想学习Python，想破解一些VIP视频，Python可以实现反爬吗？"
result = re.sub(pattern, "***", s1)
print(result)  # 我想学习Python，想***一些VIP视频，Python可以实现***吗？
```

## 07 re.split 字符串分割
### 7.1 语法
```python
re.split(pattern,string,maxsplit,flags=0)
```
### 7.2 特点
按指定正则规则分隔字符串，返回列表类型
### 7.3 示例
```python
import re
s1 = "https://www.bilibili.com/video/BV16hUaYQEsn/?spm_id_from=333.1007.tianma.1-1-1.click"
pattern = "[?|=]"
result_list = re.split(pattern, s1)
print(result_list)  # ['https://www.bilibili.com/video/BV16hUaYQEsn/', 'spm_id_from', '333.1007.tianma.1-1-1.click']
```
