# Python类的完整调用教程（零基础易懂，全案例）

你想学习 Python 中**类的调用**相关知识，这是面向对象编程的核心知识点，类的调用整体分为 **「先实例化，再调用」** 两大核心步骤，同时还有类的其他调用方式，我会从基础到进阶，用完整案例讲清楚，所有代码可直接复制运行。

## 一、前置基础：类的定义（必看）

想要调用类，首先要知道如何定义类，Python 中用关键字 `class` 定义类，语法格式：

```python
# 基础类的定义格式
class 类名():
    # 初始化方法：给对象绑定属性，创建对象时自动执行
    def __init__(self, 属性1, 属性2, ...):
        self.属性1 = 属性1
        self.属性2 = 属性2
    
    # 类的方法（类中定义的函数）
    def 方法名(self, 形参...):
        方法体逻辑
```

### 关键注意点

1. `self` 是类中**所有方法的第一个必传参数**，代表「当前的实例对象本身」，调用方法时**不需要手动传 self**，Python 会自动传入
2. 类名的命名规范：大驼峰（首字母大写，单词之间直接连写，如 `Person`、`Student`）

------

## 二、核心步骤 1：类的实例化（创建对象）

### ✅ 核心结论

**类本身不能直接调用属性 / 方法，必须先把类「实例化」生成对象，才能通过对象调用！**

> - 类：是一个「模板 / 蓝图」，定义了具备哪些属性和方法（比如：人类模板，定义了有姓名、年龄，会说话、会跑）
> - 实例 / 对象：是根据模板创建的「具体个体」（比如：小明、小红都是人类模板创建的具体对象）
>

### 实例化语法

```python
对象名 = 类名(初始化方法需要的参数)
```

### 实例演示（定义类 + 实例化）

```python
# 1. 定义一个学生类（模板）
class Student():
    # 初始化方法：给学生对象绑定 姓名、年龄、分数 3个属性
    def __init__(self, name, age, score):
        self.name = name  # self.name 是对象的属性，name是传入的参数
        self.age = age
        self.score = score
    
    # 定义类的方法：打印学生信息
    def show_info(self):
        print(f"姓名：{self.name}，年龄：{self.age}，分数：{self.score}")
    
    # 定义类的方法：判断是否及格
    def is_pass(self):
        return self.score >= 60

# 2. 类的实例化：创建具体的对象（根据模板造具体的学生）
stu1 = Student("张三", 18, 95)  # 创建第一个学生对象：张三
stu2 = Student("李四", 17, 58)  # 创建第二个学生对象：李四
```

> 此时 `stu1` 和 `stu2` 就是 `Student` 类的**实例对象**，每个对象是独立的，属性值互不影响。

------

## 三、核心步骤 2：类的核心调用（对象调用 属性 / 方法）✅

这是 Python 类**最常用、最核心**的调用方式，**99% 的场景都是这种方式**：

### 语法格式

```python
# 1. 调用对象的【属性】
对象名.属性名

# 2. 调用对象的【方法】（带括号，有参数传参数）
对象名.方法名(参数...)
```

### 完整调用案例（延续上面的 Student 类）

```python
# ============ 调用对象的属性 ============
print(stu1.name)  # 输出：张三
print(stu2.score) # 输出：58

# 修改对象的属性
stu2.score = 62   # 把李四的分数改成62
print(stu2.score) # 输出：62

# ============ 调用对象的方法 ============
stu1.show_info()  # 输出：姓名：张三，年龄：18，分数：95
stu2.show_info()  # 输出：姓名：李四，年龄：17，分数：62

# 调用有返回值的方法
print(stu1.is_pass()) # 输出：True
print(stu2.is_pass()) # 输出：True（因为分数已经改成62）
```

### 重要补充

调用方法时，**不需要给 `self` 传参**！比如 `stu1.show_info()` 直接调用即可，Python 解释器会自动把 `stu1` 这个对象传给方法的第一个参数 `self`。

------

## 四、其他调用方式 1：调用类的「无参构造」

### 什么是无参构造？

> 如果定义类时，**不写 `__init__` 初始化方法**，Python 会自动给类提供一个「默认的空初始化方法」（无参数），这种情况就可以**无参数实例化**。
>

### 语法 + 案例

```python
# 定义类，不写 __init__ 方法（无参构造）
class Teacher():
    # 定义一个方法
    def say_hello(self):
        print("大家好，我是老师！")

# 无参数实例化（无参构造调用）
tea1 = Teacher()

# 调用方法
tea1.say_hello() # 输出：大家好，我是老师！
```

> 这种方式适合「类的对象不需要绑定固定属性」的场景。

------

## 五、其他调用方式 2：直接调用「类本身」（特殊场景）

Python 中支持**直接用 类名 () 的方式调用类**，本质就是「实例化」，和上面的步骤完全一致，两种写法等价：

```python
# 方式1：先实例化存变量，再调用（推荐，可读性高，可重复使用对象）
stu = Student("王五", 19, 88)
stu.show_info()

# 方式2：直接调用类（类名()），生成匿名对象，调用方法（一次性使用）
Student("赵六", 20, 77).show_info()
```

输出结果：

```python
姓名：王五，年龄：19，分数：88
姓名：赵六，年龄：20，分数：77
```

### 适用场景

这种方式适合**只需要调用一次类的方法，用完就不需要这个对象**的场景，缺点是对象无法复用，调用一次就销毁了。

## 六、扩展：调用类的「类属性」和「类方法」（进阶必备）

除了上面的「实例属性、实例方法」，Python 中还有**类属性、类方法、静态方法**，它们的调用方式更灵活，**既可以通过「对象调用」，也可以通过「类名直接调用」**。

### 1. 类属性：属于类本身的属性

定义在类内部、所有方法外部的变量，被这个类的**所有实例对象共享**。

```python
class Car():
    brand = "比亚迪"  # 类属性：所有车对象共享这个品牌
    def __init__(self, color):
        self.color = color # 实例属性：每个对象独立拥有

# 调用方式1：类名 直接调用（推荐）
print(Car.brand) # 输出：比亚迪

# 调用方式2：对象调用
car1 = Car("白色")
print(car1.brand) # 输出：比亚迪
car2 = Car("黑色")
print(car2.brand) # 输出：比亚迪
```

### 2. 类方法：用 `@classmethod` 装饰的方法

第一个参数是 `cls`（代表类本身），**可以通过类名直接调用**，也可以通过对象调用。

```python
class Car():
    brand = "比亚迪"
    @classmethod
    def change_brand(cls, new_brand):
        cls.brand = new_brand

# 调用方式1：类名 直接调用（推荐）
Car.change_brand("特斯拉")
print(Car.brand) # 输出：特斯拉

# 调用方式2：对象调用
car1 = Car("白色")
car1.change_brand("小鹏")
print(Car.brand) # 输出：小鹏
```

### 3. 静态方法：用 `@staticmethod` 装饰的方法

没有 `self`/`cls` 强制参数，和普通函数几乎一样，**可以通过类名 / 对象直接调用**。

```python
class Tools():
    @staticmethod
    def add(a, b):
        return a + b

# 调用方式1：类名 直接调用（推荐）
print(Tools.add(10, 20)) # 输出：30

# 调用方式2：对象调用
tool = Tools()
print(tool.add(30, 40)) # 输出：70
```

## 七、完整总结（必记核心知识点）

> 1. Python 类的调用，**核心流程永远是：先实例化生成对象，再用对象调用属性 / 方法**（重中之重）；
>
> 2. 实例化语法：`对象名 = 类名(参数)`，参数是`__init__`方法中除了`self`的所有参数；
>
> 3. 调用语法：
>
>    - 实例属性：`对象名.属性名`
>    - 实例方法：`对象名.方法名(参数)`（不用传 self）
>
>    
>
> 4. 特殊调用方式：
>
>    - 无参构造：类不写`__init__`时，`对象名 = 类名()` 即可实例化；
>    - 直接调用类：`类名(参数).方法名()`，生成匿名对象，一次性使用；
>    - 类属性 / 类方法 / 静态方法：支持 `类名.xxx` 或 `对象名.xxx` 两种调用方式。
>
>    
>

### 最后附赠一个完整可运行的综合案例

```python
# 定义一个完整的类
class Animal():
    # 类属性
    type = "动物"
    # 有参初始化方法
    def __init__(self, name, food):
        self.name = name  # 实例属性
        self.food = food  # 实例属性
    
    # 实例方法
    def eat(self):
        print(f"{self.name}爱吃{self.food}")
    
    # 类方法
    @classmethod
    def change_type(cls, new_type):
        cls.type = new_type
    
    # 静态方法
    @staticmethod
    def run():
        print("所有动物都会跑")

# 1. 实例化（核心步骤）
dog = Animal("小狗", "骨头")
cat = Animal("小猫", "小鱼干")

# 2. 调用实例属性/方法
print(dog.name)  # 小狗
cat.eat()        # 小猫爱吃小鱼干

# 3. 调用类属性/类方法/静态方法
print(Animal.type)  # 动物
Animal.change_type("脊椎动物")
print(Animal.type)  # 脊椎动物
Animal.run()        # 所有动物都会跑

# 4. 直接调用类（匿名对象）
Animal("小鸟", "虫子").eat() # 小鸟爱吃虫子
```
