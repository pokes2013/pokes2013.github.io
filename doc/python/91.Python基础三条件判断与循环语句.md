## 一、IF条件判断

### 1.单向判断if

举例：如果一个人的成绩大于或等于60分，他就及格了。

```python
score = 60

if score >= 60:
    print('你及格了！')
```

### 2.双向判断：if…else…

举例：成绩大于或等于60分，显示及格；小于60分显示不及格。

```python
score = 40

if score >= 60:
    print('你及格了！')
else:
    print('你不及格')
```

### 3.多向判断：if…elif…else…

```python
score = 80

if 80 <= score < 100:
    print('你是学霸')
elif 0 <= score < 60:
    print('你不及格')
else:
    print('你及格了！')
```

### 4.if的嵌套

实例1：

```python
score = 55

if score >= 60:
    print('你及格了')
    if score >= 80:
        print('你是学霸')
    else:
        print('你成绩一般')
else:
    print('你不及格')
```

## 二、for循环语句

```python
## 打印出1-9每个数字
for i in range(1,10):   #（1,10）包前不包后1-9，不包含10
    print(i)
运行结果：
1
2
3
4
5
6
7
8
9

## 依次打印出python每个字母

for ch in 'python':     #依次打印出python每个字母
    print(ch)
    
运行结果：
p
y
t
h
o
n
```

## 三、while循环语句

```python

a=1
while a<=5:
    print(a)
    a=a+1
    
运行结果：

2
3
4
```
